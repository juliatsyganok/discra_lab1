# Задание: Приложение для работы с многочленами

## Часть 1

Разработать прикладное приложение для решения задач с применением многочленов.

### 1
Реализовать проверку принадлежности многочлена \( f(x) \) линейной оболочке, которая порождена многочленами  
\( g_1(x), g_2(x), \dots, g_k(x) \).

Если ответ положительный, получить представление вида

\[
f(x) = \sum_{i=1}^{k} A_i g_i(x).
\]

---

### 2
Для заданного многочлена \( f(x) \) и числа \( a \in \mathbb{R} \) получить его представление в виде линейной комбинации степеней

\[
(x-a)^k.
\]

---

### 3
Пусть многочлен

\[
f(x) = f_0 + f_1(x-a) + \dots + f_k(x-a)^k,
\]

где \( f_i, a \in \mathbb{R} \).

Получить его представление по степеням

\[
(x-b),
\]

где \( b \in \mathbb{R} \).

---

### 4
Рассмотрим рациональную функцию

\[
R(x) = \frac{f(x)}{g(x)},
\]

где \( f(x), g(x) \) — многочлены известной степени.

Реализовать методы нахождения

\[
\lim_{x \to A} R(x), \quad \lim_{x \to \pm\infty} R(x).
\]

---
